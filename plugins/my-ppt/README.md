# My PPT

Personal Codex plugin for PPT strategy, design, prompts, scripts, and repeatable presentation workflows.

## Layout

- `skills/my-ppt/SKILL.md`: main Codex skill and routing rules.
- `skills/my-ppt/prompts/`: reusable prompts for storyline, visual direction, and critique.
- `skills/my-ppt/workflows/`: step-by-step workflows for new decks, redesigns, and reviews.
- `skills/my-ppt/references/`: reusable lessons from past PPT production/review threads.
- `scripts/`: local helpers for turning outlines into slide plans and checking deck specs.
- `assets/`: place for templates, screenshots, icons, exported theme references, and brand examples.

## Suggested Growth

- Add PowerPoint templates under `assets/templates/`.
- Add brand systems under `assets/brands/<brand-name>/`.
- Add production scripts that generate `.pptx` files from JSON slide plans.
- Add render-and-review scripts that export slide images before delivery.

## Core Workflow

Start with `skills/my-ppt/workflows/group-meeting-deck.md` for a careful group-meeting PPT process. It is intentionally staged: finish one step, check it heavily, then decide whether to move on.

For existing HTML/PPTX production work, also read `skills/my-ppt/references/ppt-maker-reviewer-lessons.md` before editing.
