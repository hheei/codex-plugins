# Deck Critique Prompt

Use this when reviewing an outline, exported slide images, or a draft deck.

## Review Lens

- Strategic clarity
- Narrative flow
- Slide titles and takeaways
- Evidence quality
- Visual hierarchy
- Layout density
- Consistency
- Presenter readiness

## Output

List findings first, ordered by severity. For each finding, include the affected slide or section, the issue, why it matters, and a concrete fix. End with the top three changes that would most improve the deck.
