# Storyline Prompt

Use this when turning notes into a persuasive deck structure.

## Inputs

- Topic:
- Audience:
- Desired decision or action:
- Presentation length:
- Existing material:
- Tone:

## Task

Create a slide-by-slide storyline with one idea per slide. Use action titles that state the takeaway, not labels. For each slide, specify purpose, key evidence, recommended visual, and speaker intent.

## Output

- Core message
- Narrative arc
- Slide plan table
- Missing inputs
- Build risks
