# New Deck Workflow

1. Clarify the brief: audience, decision, stakes, time, format, and constraints.
2. Draft the core message in one sentence.
3. Build the narrative arc: context, tension, insight, recommendation, proof, next step.
4. Create a slide plan with action titles, purpose, evidence, and visual recommendation.
5. Define a compact visual system before producing slides.
6. Produce the deck or hand off a production-ready spec.
7. Review exported slides for hierarchy, alignment, density, and consistency.

Default deliverable: a slide-by-slide table plus production checklist.
