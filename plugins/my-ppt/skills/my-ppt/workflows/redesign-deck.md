# Redesign Deck Workflow

1. Inspect the latest deck, exported images, or outline.
2. Identify the intended audience, decision, and presenter setting.
3. Separate story problems from visual problems.
4. Preserve useful content, remove duplication, and rewrite slide titles as takeaways.
5. Apply a consistent layout system and simplify dense slides.
6. Create a before/after change list so revisions are easy to review.
7. Render or preview the deck before handoff when possible.

Default deliverable: prioritized redesign plan and revised slide structure.
