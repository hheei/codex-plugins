# Review Deck Workflow

1. Inspect the latest available deck, outline, or slide export.
2. Review story first, visuals second, production polish third.
3. Lead with high-impact findings.
4. Give concrete fixes, not vague taste notes.
5. Call out missing evidence, unclear claims, and over-dense slides.
6. End with the smallest set of changes that would noticeably improve the deck.

Default deliverable: findings list, quick wins, and next revision checklist.
